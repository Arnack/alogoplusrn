# AlgoritmPlus Mobile

Expo React Native приложение для платформы AlgoritmPlus фриланс-биржи для самозанятых.

## Описание

Мобильное приложение повторяет логику Telegram бота @Algoritmplus_bot и предоставляет следующий функционал:

### Возможности

- **Авторизация и регистрация**
  - Вход по телефону + ИНН
  - Регистрация с выбором города
  - Проверка статуса самозанятости
  
- **Поиск заказов**
  - Просмотр доступных заказов по заказчикам
  - Фильтрация по дате и смене
  - Подача заявки на заказ
  
- **Мои заказы**
  - Просмотр заявок и назначенных заказов
  - Отзыв заявки
  - Отказ от назначенного заказа
  - Подписание актов
  
- **Кошелёк**
  - Просмотр баланса
  - Создание заявки на вывод (мин. 2600₽)
  - История платежей и заявок
  
- **Профиль**
  - Личные данные (ФИО, ИНН, паспорт)
  - Управление банковской картой
  - Рейтинг и статистика
  - Реферальная программа
  - Правила работы
  
- **Уведомления**
  - Просмотр уведомлений
  - Отметка прочитанными
  
- **Акции и бонусы**
  - Активные акции и промо
  - Отслеживание прогресса
  - История бонусов

## Технологии

- **React Native** с Expo
- **TypeScript**
- **React Navigation** для навигации
- **Axios** для API запросов
- **AsyncStorage** для локального хранения
- **Clean & Strict UI** - строгий, чистый дизайн

## Установка

```bash
# Установка зависимостей
npm install

# Запуск приложения
npm start

# Запуск на iOS
npm run ios

# Запуск на Android
npm run android
```

## API Configuration

Для подключения к бэкенду измените `API_BASE_URL` в `src/constants/index.ts`:

```typescript
export const API_BASE_URL = 'https://your-api-url.com/api/v1';
```
